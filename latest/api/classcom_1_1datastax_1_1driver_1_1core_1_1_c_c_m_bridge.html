◆ getCassandraVersion() [1/2]

static VersionNumber com.datastax.driver.core.CCMBridge.getCassandraVersion ( VersionNumber  dseVersion)
static
Return
The mapped cassandra version to the given dseVersion. If the DSE version can't be derived the following logic is used:
  1. If <= 3.X, use C* 1.2
  2. If 4.X, use 2.1 for >= 4.7, 2.0 otherwise.
  3. If 5.X, use 3.0 for 5.0, 3.11 otherwise.
  4. Otherwise 4.0

The target protocolVersion to use when connecting to this CCM cluster.

This should be based on the highest protocol version that both the cluster and driver support.

For example, C* 2.0.17 should return ProtocolVersion#V2 since C* supports up to V2 and the driver supports that version.
