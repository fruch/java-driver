void com.datastax.driver.core.querybuilder.QueryBuilderTest.should_inline_custom_codec ( ) throws Exception

Ensures that a statement can be printed with and without a required custom codec. The expectation is that if the codec is not registered, then the query string should contain bind markers for all variables; if however all codecs are properly registered, then the query string should contain all variables inlined and formatted properly.
