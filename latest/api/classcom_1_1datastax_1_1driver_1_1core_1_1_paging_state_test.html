void com.datastax.driver.core.PagingStateTest.should_fail_if_paging_state_altered ( )

Validates that if the PagingState is altered in any way that it may not be reused. The paging state is altered in the following ways:

  1. Altering a byte in the paging state raw bytes.
  2. Setting the PagingState on a different Statement. (should fail hash validation)
Test Category
paging
Expected Result
PagingState refused to be reused if it is changed or used on a different statement.
