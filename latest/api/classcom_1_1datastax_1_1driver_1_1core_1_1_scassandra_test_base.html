Detailed Description

Base class for Scassandra tests. This class takes care of starting and stopping a Scassandra server, and provides some helper methods to leverage the creation of Cluster and Session objects. The actual creation of such objects is however left to subclasses. If a single cluster instance can be shared by all test methods, consider using com.datastax.driver.core.ScassandraTestBase.PerClassCluster instead.
