◆ stopDC()

void com.datastax.driver.core.ScassandraCluster.stopDC ( Cluster  cluster,
int  dc 
)

First stops each node in

dc

and then asserts that each node's Host is marked down for the given Cluster instance within 10 seconds.

If any of the nodes are the control host, this node is stopped last, to reduce likelihood of control connection choosing a host that will be shut down.

Parameters
cluster cluster to wait for down statuses on.
Parameters
