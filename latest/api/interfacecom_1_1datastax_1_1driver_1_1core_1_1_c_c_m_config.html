Detailed Description

The configuration to use when running tests with CCMTestsSupport.

This annotation can be used at the class level, or at the method level if you annotate the class with

@CreateCCM(PER_METHOD)

.

boolean [] com.datastax.driver.core.CCMConfig.createCcm ( )

Returns

true

if a CCMBridge instance should be automatically created,

false

otherwise.

If

true

, a CCM cluster builder will be created or retrieved from the cache. If

false

The cluster will be created once for the whole class if CCM test mode is CreateCCM.TestMode#PER_CLASS, or once per test method, if the CCM test mode is CreateCCM.TestMode#PER_METHOD,

boolean [] com.datastax.driver.core.CCMConfig.createKeyspace ( )

Returns

true

if a test keyspace should be automatically created,

false

otherwise.

If

true

, a cluster builder will be obtained by invoking the method specified by clusterProvider() and clusterProviderClass(), then a session will be created through Cluster#connect(), and this session will be used to create the test keyspace.

The keyspace will be created once for the whole class if CCM test mode is CreateCCM.TestMode#PER_CLASS, or once per test method, if the CCM test mode is CreateCCM.TestMode#PER_METHOD.

The test keyspace will be automatically populated upon creation with fixtures provided by testInitializer() and testInitializerClass().

Return
true
if a test keyspace should be automatically created,
false
otherwise.
