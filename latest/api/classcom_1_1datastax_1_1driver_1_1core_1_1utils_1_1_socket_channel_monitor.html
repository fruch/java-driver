Detailed Description

Utility that observes SocketChannels. Helpful for ensuring that Sockets are actually closed when they should be. Utilizes NettyOptions to monitor created SocketChannels.

void com.datastax.driver.core.utils.SocketChannelMonitor.report ( Predicate< SocketChannel >  channelFilter)

Report for all sockets matching the given predicate. The report format reflects the number of open, closed, live and total sockets created. This is logged at DEBUG if enabled.

If TRACE is enabled, each individual socket will be logged as well.

Parameters
channelFilter used to determine which sockets to report on.
