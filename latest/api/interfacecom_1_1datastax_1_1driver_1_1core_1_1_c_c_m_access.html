◆ getProtocolVersion() [1/2]

ProtocolVersion com.datastax.driver.core.CCMAccess.getProtocolVersion ( )
Return
The target protocolVersion to use when connecting to this CCM cluster.

This should be based on the highest protocol version that both the cluster and driver support.

For example, C* 2.0.17 should return ProtocolVersion#V2 since C* supports up to V2 and the driver supports that version.
